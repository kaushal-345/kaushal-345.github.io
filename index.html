Unlike basic online store with only static product listings, My Store features a fully functional "Add to Cart" system using JavaScript. Users can add, remove, and update items in their cart, view the total order value, and place orders. The project also includes user authentication, order tracking, and dynamic UI updates, making it a step beyond simple frontend design.

Social Media

My Social Media is an interactive dashboard that utilizes the JSONPlaceholder API to fetch and display user data. Initially, users select from 10 available users and sign in. Once logged in, they can navigate between different sections—Posts, Albums, Profile, and To-Do List—all dynamically retrieved from the API.
